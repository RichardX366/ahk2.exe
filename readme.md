# AHK.exe

AHK2.exe installs AutoHotKey.exe v2 and provides a path to the executable.

## Installation

Use [npm](https://www.npmjs.com) to install AHK2.exe.

```bash
npm i ahk2.exe
```

## Usage

```js
const pathToAHK = require('ahk2.exe');
// This outputs the path to AutoHotKey.exe
```
